# **App Name**: ScholarQuiz

## Core Features:

- User Authentication: Secure sign-up and login allows users to personalize their learning experience and track progress.
- Quiz Selection & Filtering: Browse and search concours exam categories and subjects to quickly find relevant quizzes from the home screen.
- Dynamic Question Generation Tool: An AI-powered tool that generates fresh practice questions tailored to selected exam categories and difficulty levels, expanding study material beyond a static database.
- Interactive Quiz Flow: Displays one question at a time with multiple-choice options, a progress bar, and immediate feedback upon answer selection.
- Personalized Performance Metrics: Tracks and displays individual user statistics, including total quizzes completed, average scores, and favorite subjects in a dedicated profile section.
- Post-Quiz Review & Analysis: Presents a summary of quiz results, including score percentage, correct/wrong answers, and a motivational message, with options to retry or go home.
- Seamless Mobile-First Navigation: Provides a responsive UI with clear and simple navigation, ensuring a smooth and intuitive user experience across mobile devices.

## Style Guidelines:

- Light color scheme, professional and calm. Primary blue: #2661B9 (HSL: 220, 65%, 45%). Background: Soft light gray with a hint of blue #E4E7ED (HSL: 220, 18%, 92%). Accent color: Vibrant cyan #4DC4E2 (HSL: 190, 70%, 60%) for interactive elements and highlights.
- Main font: 'Inter' (sans-serif) for both headlines and body text, ensuring modern, clean, and highly readable typography that is neutral yet professional and friendly.
- Use a consistent set of simple, outline-style or filled icons that are easily recognizable and reinforce the app's clean and minimal aesthetic.
- Mobile-first, responsive design with generous smooth spacing, clean visual hierarchy, and ample whitespace. All cards and buttons will feature a prominent border-radius for a soft, friendly, and modern rounded appearance.
- Incorporate subtle and simple animations, such as smooth transitions between screens and gentle feedback animations for user interactions (e.g., answering a question), to enhance user experience without distraction.
